Unreal Engine 5 Editor Utility Widget Adaptive AI Final Year Project

An adaptive boss, built for indie teams

Real-time player intent classification → dynamic boss behaviour

The tool tracks player attacks, dodges, distance, and health, then classifies the player's intent — Aggressive, Defensive, Panic, or Balanced — and adjusts the boss's behaviour accordingly. Developers can use it in Fixed mode for a single hand-picked behaviour, or Adaptive mode to let the boss respond to the player automatically.

Boss Modes
Normal, Counter, Chase Aggressive, and Heavy Pressure — each tuned to respond to a different player intent.
Intent Tuning
Adjustable attack/dodge score gain, decay rate, classification interval, and panic/balance thresholds.
Combat Configuration
Per-mode tuning of attack cooldown, attack/stop range, movement speed, and damage.
Player Integration
Drop in a pre-built player setup, or wire the tool into an existing project with auto-setup and validation tools.
Player Systems
Light/heavy combo attacks, directional dodge, sprint, weapon hit detection, and combo queueing.
Boss Systems
AI chasing and attacking, adaptive mode switching, weapon collision, and animation-driven combat.
